---
title: "免費 vs. 訂閱"
---

我們的理念是為訂閱用戶打造並提供最好的產品體驗，並同時將 Life Note、日記書寫以及 AI 導師的種種好處，分享給越多人。

Life Note 是完全靠訂閱支持成長的，我們永遠不會販賣您的資料。

|        | 免費          | 訂閱             |
| :----- | :---------- | :------------- |
| 日記     | 🟡 三篇一週     | ✅ 無限制          |
| 導師回覆   | 🟡 每篇日記一位導師 | ✅ 每篇日記最多三位導師   |
| 深度聊天   | 🟡 每天限量     | ✅ 更多使用量與深度聊天體驗 |
| 週日反思心  | 🚫          | ✅              |
| 週日內觀畫  | 🚫          | ✅              |
| 長期記憶   | 🚫          | ✅              |
| 智慧收藏   | ✅           | ✅              |
| 語音日記   | ✅           | ✅              |
| 內心天氣   | ✅           | ✅              |
| 生活對齊行動 | ✅           | ✅              |
| 目標     | 基礎的導師目標導引   | 更深入導師目標導引      |
