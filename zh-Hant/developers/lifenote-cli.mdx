你的日記，是你思考得最誠實的地方。把它連接到你已經在用的 AI — 不論是在聊天應用程式裡，還是在你的終端機中 — 讓它能夠汲取你真實的省思、智慧與目標。唯讀、私密，而且只在你開口時才會使用。

## 兩種連接方式

<CardGroup cols={2}>
  <Card title="在聊天應用程式中 — 免安裝" icon="comments">
    將 Life Note 加入 **Claude** 或 **ChatGPT**，登入一次即可。適合大多數人。
  </Card>
  <Card title="在你的終端機或程式開發代理中" icon="terminal">
    `lifenote` 命令列工具，適用於 Claude Code、Cursor 與 Codex。最適合開發者。
  </Card>
</CardGroup>

## 在 Claude 或 ChatGPT 中連接

免安裝 — 只要新增一次連接器即可。

<Steps>
  <Step title="在 Claude 中（claude.ai）">
    開啟 **設定 → 連接器 → 新增自訂連接器**，貼上下方的網址，然後點選 **連接** 並以你的 Life Note 帳號登入。
  </Step>
  <Step title="在 ChatGPT 中（Pro 及以上方案）">
    開啟 **設定 → 連接器**，以相同的網址新增一個自訂連接器，然後登入。
  </Step>
</Steps>

Life Note 連接器網址：

```
https://mcp.mylifenote.ai/mcp
```

<Note>
  新增自訂連接器可能需要付費的 Claude 或 ChatGPT 方案。它也可搭配 Gemini Enterprise 及其他相容 MCP 的助理使用。
</Note>

## 你可以問些什麼

連接完成後，只要像平常一樣說話 — 你的助理會自行決定何時查看你的日記：

> _「我最近都在寫些什麼？」_
>
> _「今年我對工作的日記中，出現了哪些模式？」_
>
> _「關於焦慮，我保存了哪些智慧？」_
>
> _「我現在正在努力的是什麼 — 背後更深層的原因又是什麼？」_
>
> _「根據我寫下的內容，關於這個決定，我該記住些什麼？」_

<Note>
  **唯讀且私密。** 你的 AI 可以*閱讀*你的日記、智慧、練習與目標。它**無法**寫入你的日記、更改你的設定，或存取任何其他人的資料。你的登入權杖以安全方式儲存 — 絕不以明文形式保存 — 而且你隨時可以在設定中中斷連接。
</Note>

## 給開發者 — CLI 工具

偏好終端機，或在程式開發代理中工作？安裝 `lifenote` CLI，在命令列中使用同一份日記。

<Steps>
  <Step title="安裝並登入">
    需要 Python 3.10 以上版本。我們推薦 [pipx](https://pipx.pypa.io/)：

    ```bash
    pipx install lifenote
    lifenote auth login
    ```
  </Step>
  <Step title="將它連接到你的程式開發代理">
    Claude Code — 執行這行指令。Cursor 與 Codex 使用相同的模式；若是 Claude Desktop，請將它加入你的 MCP 設定。

    ```bash
    claude mcp add lifenote -- lifenote mcp
    ```
  </Step>
  <Step title="教你的代理何時該使用它">
    安裝隨附的技能，讓你的代理知道何時該取用你的日記 — 以及何時不該：

    ```bash
    lifenote skills install
    ```
  </Step>
  <Step title="或直接從終端機使用">
    搜尋你的日記，並將結果導入任何代理：

    ```bash
    lifenote today
    lifenote search "anxiety about launch"
    lifenote search "burnout" --json | claude "summarize the patterns"
    ```
  </Step>
</Steps>

### 你的代理能做些什麼

14 種唯讀工具，涵蓋你的日記、保存的智慧、導師對話、練習與探索，以及人生方向。因此你可以這樣問你的代理：

> _「關於我的共同創辦人，我都寫了些什麼？」_
>
> _「總結今年我對工作的日記中的主題。」_
>
> _「我的人生目標，以及今年的重點是什麼？」_
